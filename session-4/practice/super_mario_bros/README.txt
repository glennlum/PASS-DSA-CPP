 
----About Mario.jar----


About the program:
Here's a program called Super Mario Bros Java. 
Simply download and double-click on the file to execute.

About JAR:
JAR stands for Java ARchive. It's a file format based on the popular ZIP file format and is used for aggregating many files into one.


Requirements: 
You must first have Java SE Runtime Environment installed on your machine.


You can find out more about JRE here: 
https://www.oracle.com/java/technologies/javase-jre8-downloads.html
